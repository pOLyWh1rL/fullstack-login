# Full Stack Login System

## Features
- User Registration
- Login Authentication
- Dashboard
- User List

## Tech Stack
- PHP
- MySQL
- Bootstrap 5
- XAMPP

## Setup Instructions
1. Import `database.sql` into phpMyAdmin
2. Create `config/db.php`
3. Start Apache & MySQL
4. Visit `http://localhost/fullstack-login/public`

## Security
Sensitive credentials are excluded using `.gitignore`
